9. Reports

The following is a description of EPPORA reports. A key feature of EPPORA reports is the users ability to select which information (columns) for the report is displayed along with the order of each column.

9.2 Report Pages

9.2.19 Users Report

The Users Report provides information about one or more users. Only the Users Name is required. Certain information is restricted to a finite set of users, such as hourly salary, productivity factor and cost effectiveness. These three entities are only available to the user him/herself and Executives. Each user can obtain their personal report. Executives can obtain a report about all users, while Project Managers and Project Portfolio Managers can obtain a report about project team members assigned to their projects during the current allocation period. Figure 90 shows an example of the default User Report. A user can customize a report, as he desires, by selecting/deleting different Information types.

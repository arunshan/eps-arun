6        Interface (Tabs and Commands)

Section 6 describes the “Login” and “Home” Pages along with the tabs and commands common to all EPPORA pages except the dialogues.

 

6.1    Login Page

It is necessary to login to start the EPPORA system.  Figure 3 depicts an EPPORA “Login” Page and Table 22 provides a description of each of the “Login” Page’s objects.  The “Login” Page is invoked when a user selects the EPPORA icon from the desktop.

 

Figure 3—Login Page

 

 

Object Title

Object Type

Description

Exit

Button

The “Exit” button is used to exit from EPPORA without logging in.

Login

Button

The “Login” button opens EPPORA’s “Home” Page for this user.

Password

Text Field

The “Password” is case sensitive and must be the appropriate password for the designated “User Name” to complete a successful login.  As the “Password” is typed each character is displayed as an asterisk (“*”).  A user is given three attempts to correctly specify the “User Name” and the “Password” prior to EPPORA displaying a diagnostic.

Response

Text Field

Depending on the option settings, a “User Question” may be asked along with the “User Name” and “Password.”  The response must be identical to what the user specified in his/her attributes.

User Name

Text Field

The “User Name” must be an already existing EPPORA user previously established by an Administrator.  The name is case sensitive.  A user is given three attempts to correctly specify the User Name prior to EPPORA displaying a diagnostic.

User Question

Text Field

The three possible “User Questions” are:

0=What is the name of your favorite pet?

1=What is the name of your favorite sports team?

2=What is your favorite color?

6.2    Home Page

The “Home Page” is where the major functions of EPPORA are selected.  Depending on the “User Type” of the individual that logged in, one or more of the tab commands in the “Home” Page and other EPPORA pages may be dimmed (inaccessible).  For most tab commands it is necessary to select a project from the “Project Name” drop-down list.  The “EPPORA” system returns to the Home Page after a user function is completed.  The Home Page is invoked from the Login Page.

6.3    Tabs
